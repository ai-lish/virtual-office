## Negative Prompt（避免）

```
realistic, photorealistic, 3d render, dark, scary, violent, complex background, blurry, low quality, distorted face, deformed body, bad anatomy
```

## Prompt 組合模板

```
[Cute cartoon [ANIMAL] character named [NAME], [DISTINGUISHING FEATURE]], [ACTION POSE], full body, [SETTING DESCRIPTION], K1 preschool style, clean vector art, soft pastel colors, children's educational app illustration, bright cheerful atmosphere
```
