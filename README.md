# 🏢 虛擬辦公室 - Virtual Office

日本風格的虛擬辦公室網頁，展示 AI Agents 的工作狀態。

## 🌐 網址

**https://math-lish.github.io/virtual-office/**

## ✨ 功能

### 左側導航欄 (1/3)
- 📋 項目列表 (欠交功課系統、少康網站、數學週、全方位學習日等)
- 🤖 Agents 狀態 (大腦、T仔、師弟)

### 右側虛擬辦公室 (2/3)
- 🧠 AI 區 - 大腦的工作空間
- 💻 編程區 - 師弟的開發空間  
- ☕ 休息區 - T仔的休息空間
- 👥 會議區 - 團隊協作空間

### 系統資訊
- 💻 電腦型號：Mac mini (Apple M2)
- 💾 儲存：228GB
- 🧠 RAM：8GB
- 🦀 OpenClaw Gateway：548MB

## 📱 響應式設計

- 支援手機、平板、電腦
- 自動調整佈局

## 🔄 自動刷新

每 30 秒自動刷新數據

## 🛠 技術

- HTML5 + CSS3 + JavaScript
- 日式設計風格 (日本紅、障子紙效果)
- OpenClaw API 整合

## 本地運行

```bash
# 安裝依賴
npm install

# 啟動服務器
npm start

# 訪問 http://localhost:3000
```
