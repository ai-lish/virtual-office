## 自動化流程

```
用戶錄音 → AI 編曲 → AI Tuning → AI 歌詞影片 → AI 發布 YouTube
```

### 流程說明

1. **錄音** — 用戶提供原始人聲錄音
2. **AI 編曲** — 自動生成背景音樂/編曲
3. **AI Tuning** — 人聲自動修音/調校
4. **AI 歌詞影片** — 自動生成配上歌詞的影片
5. **AI 發布** — 自動發布至 YouTube 頻道
