## 🚀 開始練習

### Step 1：進入首頁

👉 前往：https://ai-lish.github.io/ai-learning/

### Step 2：選擇試卷

1. 點擊「📝 考試專區」
2. 選擇「公開考試 (DSE)」
3. 選擇卷別：
   - **卷一 (LQ 長題目)** — 198 題長答題
   - **卷二 (MC 選擇題)** — 495 題選擇題

### Step 3：選擇年份（可選）

- 預設顯示全部年份（2012-2022）
- 可選擇特定年份如「2022」

### Step 4：選擇課題（可選）

根據卷別自動切換課題列表：

**卷一課題：** 代數、因式分解、概率、統計、三角學、幾何、圓、立體幾何、多項式、二次函數、數列...

**卷二課題：** 圓形幾何、圖形幾何、二次函數、直線方程、對數函數、排列與組合、等差數列、等比數列、軌跡...

### Step 5：開始練習

點擊「開始練習 →」進入答題頁面

## ✏️ 卷二（MC 選擇題）

### 答題流程

#### 1️⃣ 選擇答案

- 點擊 A / B / C / D 任一選項
- 選中的選項會變色（藍色邊框）

#### 2️⃣ 核對答案

- 點擊「核對答案」按鈕（綠色）
- 系統即時顯示結果：
  - ✅ **綠色** — 答對了！
  - ❌ **紅色** — 答錯了，正確答案會標記顯示

#### 3️⃣ 睇解題（可選）

- 點擊「📖 睇解題」按鈕（紫色）
- 顯示詳細解題步驟，包括：
  - 課題名稱
  - 逐步解題過程
  - 為什麼其他選項錯

#### 4️⃣ 重做類似題（答錯時）

- 答錯後會出現「🔄 重做類似題」按鈕（紫色）
- 點擊後系統根據同一課題生成新題目
- 讓學生反覆練習直到掌握

### 頁面功能

| 功能 | 說明 |
|------|------|
| 年份篩選 | 下拉選擇特定年份 |
| 課題篩選 | 下拉選擇特定課題 |
| 核對答案 | 綠色按鈕，即時顯示對錯 |
| 睇解題 | 紫色按鈕，顯示詳細步驟 |
| 重做類似題 | 紫色按鈕，答錯後出現 |
| 返回首頁 | 左上角返回按鈕 |

## 📝 卷一（長答題）

### 答題流程

#### 1️⃣ 輸入答案

- 使用鍵盤或數字鍵盤輸入答案
- 支援數學符號：× ÷ + − ( ) % /

#### 2️⃣ 提交答案

- 點擊「✓ 提交」按鈕

#### 3️⃣ 核對答案

- 點擊「核對答案」按鈕（綠色）
- 顯示官方評分標準（OCR 文字版本）

#### 4️⃣ 查看原圖（可選）

- OCR 文字為主顯示
- 點擊「🖼️ 查看原圖」可看官方答案圖片

### 頁面功能

| 功能 | 說明 |
|------|------|
| 數字鍵盤 | 支援數學符號輸入 |
| 提交答案 | 記錄你的答案 |
| 核對答案 | 顯示官方評分標準 |
| 官方答案 | OCR 文字為主，圖片可摺疊 |
| 查看原圖 | 按鈕展開官方答案圖片 |

## 🎯 仿題練習

當你在 MC 題答錯時，可以：

1. 點擊「🔄 重做類似題」
2. 系統從仿題資料庫（500+ 模板）提取相同課題的題目
3. 自動生成新數字的類似題目
4. 反覆練習直到完全掌握

**仿題特點：**
- 同一課題，不同數字
- 即時生成，無限練習
- 涵蓋主要課題：圓形幾何、指數函數、概率、等差數列等

## ❓ 常見問題

### Q1：點解有些題目冇解題？

A：部分較新年份（2021-2022）的題目仍在補充中。

### Q2：P1 長答題要點答？

A：輸入你的答案步驟，系統會顯示官方評分標準比對。

### Q3：可以篩選特定課題嗎？

A：可以！在年份選擇下方的課題下拉選單選擇你想練習的課題。

### Q4：仿題同原題一樣嗎？

A：唔同！仿題係根據相同課題不同數字生成，讓你針對性練習。

### Q5：點解我答啱咗但係顯示紅色？

A：請確認你選擇的係 A/B/C/D 中正確嘅答案。如果持續出錯，請截圖聯絡老師。

### Q6：解題步驟可以去到幾詳細？

A：目前解題包含課題名稱、關鍵定律、逐步化簡過程、以及為什麼其他選項係錯嘅。
