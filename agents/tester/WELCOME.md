
## 🆕 ClawTeam 已安裝！

ClawTeam (HKUDS 開源項目) 已安裝喺呢個系統。

### ClawTeam 功能：
- AI Agent Swarm Intelligence
- 多個agents協作
- 命令：`clawteam spawn`, `clawteam launch`

### 安裝位置：
- Repo: ~/.openclaw/workspace/ClawTeam/
- CLI: `clawteam --help`
- Python: `python3.11 -m clawteam`

### 使用方法：
```bash
# Create a team
clawteam team spawn-team my-team

# Launch swarm
clawteam launch --template default "task"
```
