## 詳細製作計劃

### 1. 書記浮動頭像優化 🎭
**目標**: 加入更多表情變化

**具體任務**:
- [ ] 增加 5 種心情狀態 (開心、思考、忙碌、休息、驚訝)
- [ ] 根據工作狀態自動切換表情
- [ ] 加入表情動畫過渡效果
- [ ] 在浮動頭像 tooltip 顯示當前心情

**相關檔案**:
- `index.html` - 書記浮動頭像區塊

### 2. 工作氣泡動畫改進 💬
**目標**: 加入彈性動畫效果

**具體任務**:
- [ ] 使用 CSS `bounce` 動畫
- [ ] 加入彈性過渡效果 (elastic transition)
- [ ] 根據任務緊急性質顯示不同動畫速度
- [ ] 加入完成時慶祝動畫
