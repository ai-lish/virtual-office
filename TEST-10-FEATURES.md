## 測試方法

1. 打開瀏覽器開發者工具 (F12)
2. 進入 Console 標籤
3. 執行各項功能測試
4. 檢查輸出和界面變化

---

## 預期輸出範例

```javascript
// 測試書記心情
updateSecretaryMood();
// 輸出: 在正確位置添加 mood emoji

// 測試通知分類
addNotification('測試', '這是一條測試通知', 'task');
// 輸出: 在通知列表添加帶分類的項目

// 測試數據導出
exportData('json');
// 輸出: 下載 JSON 檔案

// 測試權限
setUserRole('admin');
console.log(hasPermission('export'));
// 輸出: true
```
